# DemoApp
DemoApp for Trying out Xunit

A unit test is written for a unit of code → typically an individual method inside a class.

- **[Fact]**  simple test

-  **[Theory]**  parameterized test  >>   [InlineData] → provides test data for [Theory].
  
   [InlineData(2, 3, 5)]
   
   [InlineData(10, 20, 30)]
   
   [InlineData(-5, 5, 0)]
